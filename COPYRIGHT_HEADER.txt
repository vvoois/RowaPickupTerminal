 * PROJECT OVERVIEW
 * ================
 * RowaPickupSlim is a native Windows GDI application that replaces a MAUI
 * cross-platform app for warehouse robot picking operations. It communicates
 * with Baxter Rowa automation systems via WWKS 2.0 protocol over TCP/XML.
 * 
 * ARCHITECTURE
 * ============
 * - Modular design with headers declaring contracts, main.cpp implementing
 * - NetworkClient class handles TCP/WWKS protocol with auto-reconnection
 * - GDI-based UI with custom scrollbar and real-time search
 * - Thread-safe global state with mutex protection
 * - Multi-language support (English & Dutch)
 * - Comprehensive audit logging to C:\ProgramData\RowaPickup\Protocol\
 * 
 * BUILD
 * =====
 * Visual Studio 2022 | Windows SDK 10.0+ | C++17
 * 
 * BUILD: msbuild RowaPickupSlim.sln /p:Configuration=Release /p:Platform=x64
 * 
 * CONFIGURATION
 * =============
 * Settings loaded from: C:\ProgramData\RowaPickup\RowaPickupMaui.config
 * Format: INI with sections [Connection], [UI], [Security]
 * 
 * See README.md for full documentation.
